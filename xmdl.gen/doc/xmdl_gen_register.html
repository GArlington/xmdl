XMDL Platform Registry

Identifier:
xmdl.gen.registry

Since:
3.1

Description:
XMDL Platform Registry

Configuration Markup:

<!ELEMENT extension (platform+)>

<!ATTLIST extension

point CDATA #IMPLIED

id    CDATA #IMPLIED

name  CDATA #IMPLIED>

  • id - Unique extension id
  • name - Name of extension

<!ELEMENT platform EMPTY>

<!ATTLIST platform

name    CDATA #REQUIRED

version CDATA #REQUIRED

class   CDATA #REQUIRED>

  • name - Name of the platform
  • version - Platform version
  • class - Implementation class name

Examples: 

  
<extension point=
"org.xmdl.xyz.platform"
>

   
<platform
    id=
"org.xmdl.xyz.platform1"

    name=
"Sample Platform 1"

    class=
"org.xmdl.xyz.platform.SamplePlatform1"
>

   
</platform>

  
</extension>
